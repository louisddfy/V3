Corrigé des Questions

1. Quel est le rôle principal d'un système d'exploitation ?

Réponse correcte : Gérer les ressources matérielles et logicielles

2.Quel protocole est principalement utilisé pour envoyer des emails ?

Réponse correcte : SMTP

3. Que signifie SQL ?

Réponse correcte : Structured Query Language

4. Dans un réseau informatique, que signifie l'adresse IP 192.168.1.1 ?

Réponse correcte : Une adresse privée

5. Quel composant stocke les données à court terme pour un accès rapide par le processeur ?

Réponse correcte : RAM

6. Quel est le rôle principal d'un pare-feu (firewall) ?

Réponse correcte : Bloquer les connexions réseau non autorisées

7. Quel langage est principalement utilisé pour le développement d’applications Android ?

Réponse correcte : Java/Kotlin

8. Quelle est la différence principale entre HTTP et HTTPS ?

Réponse correcte : HTTPS chiffre les communications avec TLS/SSL

9. Quelle structure de données fonctionne selon le principe "Last In, First Out" (LIFO) ?

Réponse correcte : Une pile (stack)

10. Quel outil Windows permet d’analyser la performance du système en détail ?

Réponse correcte : Analyseur de performance
